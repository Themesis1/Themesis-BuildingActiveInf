## About This Repository

This repository contains the Python lab code for *Building Active Inference in Python* — a hands-on lab course that runs alongside the T3 conceptual course. T3 teaches you *what* active inference is. This lab course teaches you *how to build it* in Python.

Every line of code in this course connects directly to a mathematical operation. Nothing is hidden in a black box. Students who complete this course will understand what active inference libraries do internally, because they have built the equivalent themselves.
